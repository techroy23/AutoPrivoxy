# AutoPrivoxy
Automatic Installer for Privoxy

## Usage
`````
curl -O https://raw.githubusercontent.com/techroy23/AutoPrivoxy/master/AutoPrivoxy.sh

chmod +x AutoPrivoxy.sh

./AutoPrivoxy.sh
`````
